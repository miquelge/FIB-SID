Plataforma agents i interfície:
java -cp ./lib/jade.jar:./dist/JadeApplication.jar jade.Boot -local-host 127.0.0.1 -gui &

Exemples d'execició d'Agents:
java -cp ./lib/jade.jar:./dist/JadeApplication.jar jade.Boot -local-host 127.0.0.1 -container HelloTio:org.upc.edu.Behaviours.HelloWorldAgent\("Que","pasa","tio"\) &
java -cp ./lib/jade.jar:./dist/JadeApplication.jar jade.Boot -local-host 127.0.0.1 -container CyclicAgentTest:org.upc.edu.Behaviours.CyclicBehaviourAgent\ &
java -cp ./lib/jade.jar:./dist/JadeApplication.jar jade.Boot -local-host 127.0.0.1 -container OneShotAgentTest:org.upc.edu.Behaviours.OneShotBehaviourAgent\ &
java -cp ./lib/jade.jar:./dist/JadeApplication.jar jade.Boot -local-host 127.0.0.1 -container TickerAgentTest:org.upc.edu.Behaviours.TickerBehaviourAgent\ &
java -cp ./lib/jade.jar:./dist/JadeApplication.jar jade.Boot -local-host 127.0.0.1 -container WakerAgentTest:org.upc.edu.Behaviours.WakerBehaviourAgent\ &

mira docu jena per especificar el contenidor:
    o -container <nom> (?)
    o -<nom> (?)
